# Sistema de Gerenciamento de Tarefas (Python)

Este é um projeto simples de console criado para treinar Python.  
Ele permite:

- Adicionar tarefas  
- Listar tarefas  
- Marcar como concluída  
- Excluir tarefas  
- Salvar tudo automaticamente em um arquivo `.txt`
